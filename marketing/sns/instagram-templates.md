# ROOTS Instagram 投稿テンプレート（確定版）

投稿制作に必要な素材を一括で生成するためのテンプレート集。

---

## 使い方

```
instagram-templates.mdとtone_guide.mdを読んで、
味の設計図の投稿を作ってください。
県: [県名]、食材: [食材名]、型: [情景型/記憶型/事実型/対比型/体感型/声型]
```

---

## 共通ルール

### ビジュアル
- 背景: #090909
- アクセント暖色: #F0A032（オレンジ）
- アクセント寒色: #50A0F0（ブルー）
- テキスト: #e0e0e0
- アスペクト比: 4:5（1080x1350px）
- 人の顔は映さない（手元・後ろ姿はOK）

### フォント
- 日本語本文: Shippori Mincho
- ロゴ・英語: Bebas Neue
- 数値・補足: Space Mono
- ロゴは最終スライドのみ

### 投稿文
- tone_guide.md準拠
- 敬語ベース、体言止め可
- 禁止: お得・限定・ゲット・超・やばい・神・!!
- 主語は「あなた」「この味」「ルーツ」。ROOTSを主語にしない
- 「○○の人なら知っている」のようなよいしょ表現は禁止

## テンプレート: 味の設計図

### 4枚の役割

| スライド | 役割 | 内容 |
|---------|------|------|
| 1 | HOOK（止める） | 県×感情。6型から選択。スクロールを止める |
| 2 | EDUCATE（学ばせる） | 食材がビールに入ると味がどう変わるか |
| 3 | IMAGINE（欲しがらせる） | 完成ビールの想像。スタイルとの組み合わせ |
| 4 | BRAND（覚えさせる） | ROOTSブランド接続 + ロゴ |

---

### スライド1 — HOOK（6型テンプレート）

#### 情景型（配分35%）
土地の風景を描写する。映像が脳内に浮かぶことを最優先。

**テキスト配置**:
```
中央: Shippori Mincho / 32px / #e0e0e0
→ [土地の情景を描写する1〜2文]

下部: Space Mono / 14px / #50A0F0 / 60%透過
→ FLAVOR BLUEPRINT vol.[番号]
```

**画像プロンプト**:
```
A close-up, moody still life photograph of [食材の英語名].
Shot at 45 degrees on a dark slate surface.
Single warm light source from the left creating dramatic shadows.
Color palette: deep black background (#090909), warm orange (#F0A032) rim light, cool blue (#50A0F0) fill light.
Style: premium food photography, editorial, dark and luxurious.
Droplets of water on the ingredient for freshness.
Aspect ratio: 4:5 (1080x1350px).
No text, no logos, no people.
```

**例**:
- 「富良野の夏。畑を歩くと、甘い香りがする。」（メロン×北海道）
- 「冬の福岡。あまおうの赤が、市場に並ぶ朝。」（あまおう×福岡）
- 「六月の山形。さくらんぼが枝をたわませている。」（さくらんぼ×山形）

---

#### 記憶型（配分25%）
読み手の過去の記憶を呼び起こす。問いかけではなく、記憶の断片を置く。

**テキスト配置**: 情景型と同じ

**例**:
- 「柚子の皮を割ったとき、台所に広がった香り。まだ残っている。」（柚子×高知）
- 「帰省するたびに、冷蔵庫に入っていた。」（みかん×愛媛）
- 「母が送ってくれた段ボール。開けると、あの匂いがした。」（りんご×青森 等）

---

#### 対比型（配分15%）
都会と地方、現在と過去の対比で「欠落感」を刺激する。

**テキスト配置**: 情景型と同じ

**例**:
- 「東京で『山椒』を探しても、この痺れには出会えない。」（山椒×和歌山）
- 「東京のスーパーでは、この色のみかんに出会えない。」（みかん×愛媛）
- 「毎日飲むコーヒーは選べるのに、故郷の味は選べない。」（汎用）

---

#### 事実型（配分10%）
意外な事実やデータで知的好奇心を刺激する。

**テキスト配置**: 情景型と同じ

**例**:
- 「日本のレモンの6割は、瀬戸内のこの島で育つ。」（レモン×広島）
- 「標高1,500m。わさびが育つには、この冷たさがいる。」（わさび×長野）
- 「年間降水量3,000mm。その雨が、この茶葉を育てた。」（茶×静岡）

---

#### 会話/声型（配分10%）
実際のセリフや心の独白を提示する。方言を自然に入れられる。

**テキスト配置**:
```
中央: Shippori Mincho / 32px / #e0e0e0
→ 「[セリフ]」（鉤括弧付き）
→ または [独白的な一文]

下部: Space Mono / 14px / #50A0F0 / 60%透過
→ FLAVOR BLUEPRINT vol.[番号]
```

**例**:
- 「『やっぱりこっちのは違うんよね。』」（方言系）
- 「『お土産、何がいい？』——いつもそう聞かれる。」（帰省系）
- 「冬の愛媛。こたつの上に、みかんが山になっている。」（情景+記憶の複合）

---

#### 体感型（配分5%）
五感の身体的体験を描写する。食材の味覚的特徴が極めて強い場合のみ。

**テキスト配置**: 情景型と同じ

**例**:
- 「噛んだ瞬間、甘さが弾ける。宮崎のマンゴー。」（マンゴー×宮崎）
- 「口に入れた瞬間、舌が痺れる。紀州の山椒。」（山椒×和歌山）
- 「皮をむいたとき、指先が黄色くなる。」（柑橘系）

### スライド2 — EDUCATE

**画像**: テキスト + 図解で構成（AI生成ではなくデザイン）

**テキスト配置**:
```
上部: Shippori Mincho / 28px / #e0e0e0
→ [食材名]がビールに入ると

中央: 味の変化を箇条書き
→ 香り: [変化の説明]
→ 苦味: [変化の説明]
→ 酸味: [変化の説明]
→ 甘味: [変化の説明]
各項目: Shippori Mincho / 20px / #e0e0e0
ラベル: Space Mono / 14px / #F0A032

背景: #090909
装飾: 左に細いライン / #50A0F0 / 1px
```

**1枚目で地域名を使った場合**: スライド2の冒頭に県名を回収する。
例: 「北海道のメロンがビールに入ると」


## テンプレート: リール

### 共通ルール
- 15〜30秒
- シネマティックな映像
- BGM: ローファイ、アンビエント、ピアノ系
- テキスト: Shippori Mincho + Bebas Neue
- 最後にROOTSロゴ
- 9:16（縦型）

### E-1. 食材スポットライト（15秒）

フィード投稿のダイジェスト。食材のビジュアルで引き込む。

```
0-3秒: 食材のクローズアップ映像（テキストなし）
3秒: 「[食材名]をビールに入れると」
8秒: 「[味の変化を1文で]」
12秒: 「飲んでみたいですか。」
14秒: ROOTS ロゴ
```

**映像プロンプト**:
```
A slow-motion close-up of [食材 in English] being gently placed or sliced on a dark surface.
Water droplets or juice splashing subtly.
Warm orange (#F0A032) key light, cool blue (#50A0F0) fill.
Dark, cinematic, premium feel. Shallow depth of field.
No people's faces. 9:16 vertical video.
```

### E-2. 故郷ノスタルジア（20秒）

県の風景で感情を動かす。新規リーチ獲得の主力リール。

```
0-3秒: 県の風景映像（テキストなし）
5秒: 「この土地の味、」
7秒: 「まだビールになっていません。」
12秒: 「[県の名産品に触れる一文]」
18秒: ROOTS ロゴ
```

**映像プロンプト**:
```
Cinematic aerial or ground-level footage of [県の風景 in English].
[具体的な描写].
Golden hour or blue hour lighting.
Nostalgic, melancholic beauty. Slow camera movement.
No people's faces. 9:16 vertical.
```

### E-3. ブランドメッセージ（15秒）

ROOTSの世界観を映像で伝える。

```
0秒: 黒背景
2秒: 「[印象的な一文]」フェードイン
6秒: 「[エピソード的な一文]」フェードイン
10秒: 「ルーツが誇りになるように。」フェードイン
13秒: ROOTS ロゴ
```

## 生成フロー（1投稿分）

1. **テーマ決定**: 県 + 食材 + 型（情景/記憶/事実/対比/体感/声）を決める
2. **投稿文生成**: テンプレートに沿ってtone_guide.md準拠で作成
3. **スライド1〜4の画像プロンプト生成**: 各スライドのテンプレートの変数を埋めて出力
4. **テキスト配置指示の生成**: 各スライドに重ねる文言・フォント・色・配置を出力
5. **ハッシュタグ**: 固定タグ + カテゴリ別タグを組み合わせ
6. **レビュー**: tone_guide.mdとの整合性を最終チェック
