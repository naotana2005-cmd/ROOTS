# ROOTS Instagram エージェントチーム運用ガイド

## 概要

Instagram投稿を効率的に制作するために、
Claude Codeの「カスタムスキル」を使ってエージェントチームを構築する。

`/instagram-post-create` と打つだけで、
1投稿に必要な全素材が一括生成される。

---

## チーム構成（3つの専門スキル + 1つの統合スキル）

```
/instagram-post-create（統合スキル）
  │
  ├─ Phase 1: ワード選定
  │   → ハッシュタグ・キーワード調査
  │
  ├─ Phase 2: コピーライティング
  │   → 投稿文・スライドテキスト生成
  │
  └─ Phase 3: ビジュアルディレクション
      → 画像生成プロンプト・テキスト配置指示
```

---

## 使い方

### 基本

```
/instagram-post-create
県: 北海道
食材: メロン
型: 情景型
```

### 出力される内容

1. **投稿文**（tone_guide.md準拠、ハッシュタグ付き）
2. **スライド1〜4の画像生成プロンプト**（そのままAI画像生成ツールに入力可能）
3. **スライド1〜4のテキスト配置指示**（フォント・色・サイズ・配置位置）
4. **ハッシュタグリスト**（固定タグ + カテゴリ別タグ）

---

## ファイル構成

```
ROOTS/.claude/skills/
└── instagram-post-create/
    └── SKILL.md     ← 全工程を統合するメインスキル
```

---

## 仕組みの説明（技術的でない人向け）

### カスタムスキルとは？

「レシピ本」のようなものです。

- 毎回「まずハッシュタグを調べて、次に投稿文を書いて、画像プロンプトを作って...」と指示する代わりに
- 「/instagram-post-create」と打つだけで、全工程が自動で実行される
- レシピ（手順）は `.claude/skills/` に保存されている

### なぜスキルが最適か？

| 方法 | 例えると | ROOTSに合う理由 |
|------|---------|---------------|
| カスタムスキル | レシピ本 | 毎回同じ手順で投稿を作る。最もコスト効率が良い |
| サブエージェント | 順番に専門家に相談 | 柔軟だが毎回指示が必要 |
| エージェントチーム | チームで会議 | 高コスト。定型作業には過剰 |

Instagram投稿は手順が決まっているので、「レシピ本」が最適。
