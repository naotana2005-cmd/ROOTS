# ROOTS Instagram エージェントチーム運用ガイド

## 概要

Instagram投稿を効率的に制作するために、
Claude Codeの「カスタムスキル」を使ってエージェントチームを構築している。

`/instagram-post-create` と打つだけで、
1投稿に必要な全素材が一括生成される。

---

## チーム構成

```
/instagram-post-create（統合スキル）
  │
  ├─ Phase 1: ワード選定
  │   → ハッシュタグ15〜20個を選定
  │
  ├─ Phase 2: コピーライティング
  │   → S1〜S4テキスト + キャプション生成
  │   → 改行位置・文字数を明示
  │
  └─ Phase 3: ビジュアルディレクション
      → 画像生成プロンプト（英語・2枚分）
      → px座標テキスト配置指示（4枚分）
```

---

## 使い方

### 基本

```
/instagram-post-create
県: 北海道
食材: メロン
型: 情景型
```

入力を省略した場合、`posting-calendar.md` から次の投稿を自動選択する。

### 出力される内容

1. **キャプション（投稿文）** — tone_guide.md準拠、ハッシュタグ付き
2. **スライド1〜4のテキスト** — 改行位置・文字数付き
3. **画像生成プロンプト** — S1（食材写真）とS3（ビールグラス）の2枚分。英語。そのまま画像生成AIに入力可能
4. **テキスト配置指示** — 全4枚分。px座標・フォント・サイズ・色・改行位置をすべて明記
5. **ハッシュタグリスト** — 固定タグ5個 + カテゴリ別タグ10〜15個
6. **チェックリスト** — 文字数制限・トーン・改行・配置の最終確認

## 仕組みの説明（技術的でない人向け）

### カスタムスキルとは？

「レシピ本」のようなもの。

- 毎回「まずハッシュタグを調べて、次に投稿文を書いて...」と指示する代わりに
- `/instagram-post-create` と打つだけで、全工程が自動で実行される
- レシピ（手順）は `.claude/skills/` に保存されている

### なぜスキルが最適か？

| 方法 | 例えると | 特徴 |
|------|---------|------|
| **カスタムスキル** | **レシピ本** | **毎回同じ手順で投稿を作る。最もコスト効率が良い** |
| サブエージェント | 順番に専門家に相談 | 柔軟だが毎回指示が必要 |
| エージェントチーム | チームで会議 | 高コスト。定型作業には過剰 |

Instagram投稿は手順が決まっているので、「レシピ本」が最適。

### スキルの中身がやっていること

1つのスキルの中に3人の「専門家」の手順が入っている:

```
ワード選定のプロ → どのハッシュタグが効果的か
コピーライターのプロ → どの言葉が刺さるか、どこで改行するか
ビジュアルディレクターのプロ → どこにテキストを置くか、どんな画像を生成するか
```

これらを1つのスキルに統合しているので、一回の指示で全部出てくる。
