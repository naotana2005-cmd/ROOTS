# ROOTS Instagram戦略（確定版）

## 基本方針

- **目的**: 投票開始時に即動くフォロワーを育てる
- **頻度**: 週2〜3回（月8〜10投稿）+ リール月3〜4本
- **形式**: カルーセル投稿（4枚/投稿）+ リール（15〜30秒）
- **素材**: AI生成画像が中心（MVP前のため実物素材なし）
- **トーン**: tone_guide.md準拠。静かで芯がある。郷愁と誇り
- **ターゲット**: 出身地を離れた20〜40代の日本人

## フェーズ設計

### Phase A: 種まき期（今〜投票開始まで / 約3ヶ月）

投票はまだ始まっていない。この期間の目的は3つ：
1. 「県名 × 食材 × ビール」の教育でフォロワーを集める
2. ROOTSの世界観を好きになってもらう
3. 「何か始まるらしい」という期待感を育てる

**月ごとの温度感**:

| 月 | テーマ | ROOTS露出度 | CTA |
|----|--------|------------|-----|
| 1ヶ月目 | 感情×教育。県の食材とビールの可能性 | 低め。教育コンテンツが主役。ロゴのみ | フォローを促す |
| 2ヶ月目 | 教育＋「もしこの味がビールになったら」 | 中。投稿文で自然に問いかける | フォロー + コメント |
| 3ヶ月目 | カウントダウン＋コンセプト全面公開 | 高。「投票でビールを作る」を明言 | 投票開始の告知 |

### Phase B: 収穫期（投票開始後）

育てたフォロワーを投票参加者に転換する。
スライド4がすべて投票CTAに切り替わる。
（Phase Bの詳細は投票開始1ヶ月前に策定）

## スライド1の表現型（6型ミックス戦略）

全投稿が同じ型だとマンネリ化する。
食材の特性に合わせて6つの型を使い分ける。

### 6型の定義

| 型 | 定義 | 例 | 配分 |
|----|------|-----|------|
| 情景型 | 土地の風景を描写する | 「富良野の夏。畑を歩くと、甘い香りがする。」 | 35% |
| 記憶型 | 読み手の過去の記憶を呼び起こす | 「帰省するたびに、冷蔵庫に入っていた。」 | 25% |
| 対比型 | 都会と地方、現在と過去を対比する | 「東京のスーパーでは、この色のみかんに出会えない。」 | 15% |
| 事実型 | 意外な事実で知的好奇心を刺激する | 「日本のレモンの6割は、この島で育つ。」 | 10% |
| 会話/声型 | 実際のセリフや心の独白を提示する | 「『やっぱりこっちのは違うんよね。』」 | 10% |
| 体感型 | 五感の身体的体験を描写する | 「口に入れた瞬間、舌が痺れる。」 | 5% |

### 型の配置ルール

1. **同じ型を3投稿以上連続させない**
2. **最初の5県は情景型で統一**（世界観の確立が最優先）
3. **6県目以降からミックス開始**
4. **帰省シーズン（年末年始・お盆）は記憶型・対比型・声型の比率を上げる**
5. **新生活シーズン（3〜4月）は対比型を増やす**

## スライド1の地名ルール

県名・地域名・地名なしの使い分け:

| 食材→県の連想 | 1枚目の推奨 | 例 |
|---|---|---|
| 自明（95%がわかる） | 地名なし。食材名だけで入る | シークヮーサー→沖縄、りんご→青森 |
| 連想可能だが複数県ある | 地域名で絞り込む | メロン→「富良野の」、わさび→「安曇野の」 |
| 特定困難 | 県名で入る | いちご→「栃木の」、米→「新潟の」 |

**補足ルール**: 1枚目で地域名を使った場合、スライド2の冒頭で県名を必ず回収する。

## 投稿文のフェーズ変化

### 1ヶ月目: 感情重視。ROOTSをあまり語らない

```
富良野の夏。畑を歩くと、甘い香りがする。

あの赤肉メロン。
果汁が溢れる瞬間の甘さを、まだ覚えていますか。

このメロンがビールに入ると、
芳醇な香りがグラスの中に広がります。
ヴァイツェンの柔らかいボディと出会えば、
なめらかで贅沢な一杯に。

北海道の寒暖差が育てた糖度ごと、ビールに入れる。

保存して、いつかこの味に出会う日のために。
```

### 2ヶ月目: 問いかけを混ぜ始める

```
柚子の皮を割ったとき、台所に広がった香り。
まだ残っている。

高知の山間部で育つ柚子。
寒暖差が、皮に香りを閉じ込めます。

この柚子がビールに入ると、
リモネンの香気がグラスの中で弾ける。
ペールエールの柑橘香と共鳴すれば、
和と洋が出会う、鮮やかな一杯に。

あなたの県の味も、ビールにしてみたいと思いませんか。
```

### 3ヶ月目: ROOTSのコンセプトを明示

```
日本のレモンの6割は、瀬戸内のこの島で育つ。

広島県・生口島。
潮風と太陽が育てたレモンは、皮まで食べられます。

このレモンがビールに入ると、
爽やかな酸味とほのかな苦味が重なります。
ゴーゼの塩味と出会えば、
瀬戸内の海風を飲むような一杯に。

まもなく、この味をあなたの投票で選べるようになります。
保存して、その日を待っていてください。
```

## 運用ルール

- 投稿文はtone_guide.md準拠。タメ口・「!!」・勧誘口調は禁止
- AI生成画像は毎回同じスタイルで統一感を出す（instagram-templates.md参照）
- 4枚カルーセルは必ず「1枚目で止める」ビジュアルを意識する
- リールは最初の1秒で引き込む
- 他アカウントへのコメント・いいねも積極的に行う（地方系・クラフトビール系）
- 投稿後24時間以内のコメントには必ず返信する
- フィードの3列グリッドで見たとき、トーンと色味が統一されていること
