# Instagram投稿一括生成スキル

Instagram投稿に必要な全素材を一括生成する。

## 入力

ユーザーから以下の情報を受け取る（未指定の場合はposting-calendar.mdから次の投稿を自動選択）:

- 県名
- 食材名
- 型（情景型/記憶型/事実型/対比型/体感型/声型）

## 実行手順

### 事前準備

以下のファイルを必ず読み込む:
1. `marketing/sns/instagram-templates.md` — テンプレート
2. `brand/tone_guide.md` — トーンガイド
3. `marketing/sns/instagram-strategy.md` — 戦略
4. `marketing/sns/posting-calendar.md` — カレンダー（投稿番号の確認）

### Phase 1: ワード選定

1. 県名・食材名に関連するハッシュタグを選定する
2. 固定タグ5個 + カテゴリ別タグ5〜10個を組み合わせる
3. 合計15〜20個に収める

### Phase 2: コピーライティング

以下をtone_guide.mdに準拠して生成する:

1. **スライド1テキスト**: 指定された型に従って作成
   - 情景型: 土地の風景描写
   - 記憶型: 過去の記憶の断片
   - 事実型: 意外なデータ
   - 対比型: 都会との対比
   - 体感型: 五感の描写
   - 声型: セリフや独白
   - **禁止**: 「○○の人なら知っている」のようなよいしょ表現

2. **スライド2テキスト**: 食材がビールに与える変化（香り・苦味・酸味・甘味）
   - 1枚目で地域名を使った場合、冒頭で県名を回収する

3. **スライド3テキスト**: 相性の良いビアスタイルと完成形の想像

4. **スライド4テキスト**: フェーズに応じたCTA
   - 投票開始前: 「保存して、いつかこの味に出会う日のために。」等
   - 投票開始後: 「あなたの投票で選べます。」等

5. **投稿文**: テンプレートに沿って作成。ハッシュタグ付き

### Phase 3: ビジュアルディレクション

各スライドについて以下を生成する:

1. **画像生成プロンプト**（英語）
   - instagram-templates.mdのプロンプトテンプレートの変数を埋める
   - 背景: #090909、暖色: #F0A032、寒色: #50A0F0
   - アスペクト比: 4:5
   - 人の顔は映さない

2. **テキスト配置指示**
   - フォント名・サイズ・色・配置位置を明記
   - 日本語: Shippori Mincho
   - 英語/ロゴ: Bebas Neue
   - 数値/補足: Space Mono

### 最終出力

以下の形式で出力する:

```
## [県名] × [食材名]（[型名]）

### 投稿文
[完成した投稿文。ハッシュタグ付き]

### スライド1: HOOK
画像プロンプト: [英語プロンプト]
テキスト配置: [配置指示]

### スライド2: EDUCATE
画像: テキスト+図解（デザイン制作）
テキスト配置: [配置指示]

### スライド3: IMAGINE
画像プロンプト: [英語プロンプト]
テキスト配置: [配置指示]

### スライド4: BRAND
テキスト配置: [配置指示]

### ハッシュタグ
[完成したハッシュタグリスト]
```

### レビュー

出力後、以下をセルフチェックする:
- tone_guide.mdの禁止語が含まれていないか
- 「○○の人なら」等のよいしょ表現がないか
- スライド1が感情ファーストになっているか
- 地域名を使った場合、スライド2で県名を回収しているか
- ハッシュタグが15〜20個の範囲か
