# Instagram投稿一括生成スキル

Instagram投稿に必要な全素材を一括生成する。

## 入力

ユーザーから以下の情報を受け取る（未指定の場合はposting-calendar.mdから次の投稿を自動選択）:

- 県名
- 食材名
- 型（情景型/記憶型/事実型/対比型/体感型/声型）

## 実行手順

### 事前準備

以下のファイルを必ず読み込む:
1. `marketing/sns/instagram-templates.md` — テンプレート・px座標仕様
2. `brand/tone_guide.md` — トーンガイド
3. `marketing/sns/instagram-strategy.md` — 戦略・フェーズ判定
4. `marketing/sns/posting-calendar.md` — カレンダー（vol番号の確認）

### Phase 1: ワード選定

1. 県名・食材名に関連するハッシュタグを選定する
2. 固定タグ5個 + カテゴリ別タグ10〜15個を組み合わせる
3. 合計15〜20個に収める

**固定タグ（毎回同じ）**:
```
#ROOTS #親愛なる故郷へ #ルーツビール #クラフトビール #ビール好きな人と繋がりたい
```

### Phase 2: コピーライティング

以下をtone_guide.mdに準拠して生成する:

1. **スライド1テキスト**: 指定された型に従って作成
   - **上限30文字**
   - **改行位置を必ず明示する**（句点「。」で改行、1行15文字目安）
   - **禁止**: 「○○の人なら知っている」のようなよいしょ表現

2. **スライド2テキスト**: 食材がビールに与える変化
   - **上限100文字**
   - 見出し + 4項目（AROMA / BITTER / SOUR / SWEET）
   - 各項目の説明は15文字以内
   - 1枚目で地域名を使った場合、冒頭で県名を回収する

3. **スライド3テキスト**: 相性の良いビアスタイル
   - **上限75文字**
   - サブ見出し + スタイル名（英語） + 説明1〜2行
   - 説明は1行15文字目安、最大2行

4. **スライド4テキスト**: フェーズに応じたCTA + ROOTSロゴ
   - **上限30文字**（ロゴ除く）
   - instagram-strategy.mdのフェーズに応じて選択

5. **投稿文**: テンプレートに沿って作成
   - 「[県名]を一杯のビールにするなら」の視点を自然に含める
   - 既存ビールとの差別化は投稿文内で行う（画像には入れない）
   - 「これが、[県名]の一杯の設計図のひとつ。」で締める（「のひとつ」を必ず入れる）
   - **食材を断定しない**: 「たとえば」「もし」「あなたなら」等を使い、投票で決まることを暗示する
     - ✅ 「たとえば、富良野のメロン。」「もしこの味がビールに入ったら。」
     - ❌ 「北海道のビールはメロンで作ります。」「メロンに決まり。」
   - CTAで読者の選択を促す: 「あなたなら、どの味を選びますか。」
   - ハッシュタグ付き

### Phase 3: ビジュアルディレクション

#### 3-1. 画像生成プロンプト（英語）

instagram-templates.mdのプロンプトテンプレートの変数を埋めて出力。
スライド1（食材写真）とスライド3（ビールグラス）の2枚分。

#### 3-2. 全スライドのpx座標テキスト配置指示

instagram-templates.mdのpx座標仕様に従い、各スライドの配置を出力する。
**テキストの改行位置・各行のY座標を必ず明示する。**

### Phase 4: ビール専門家チェック

Phase 2・3の出力に対して、ビール醸造の専門知識の観点からチェックする。

**チェック項目**:
1. **ビアスタイルと副原料の相性**: 選択したスタイルとその食材の組み合わせが醸造学的に妥当か
   - 例: メロン × ヴァイツェン → ◎（フルーツ系はヴァイツェンの酵母と相性が良い）
   - 例: 柚子 × スタウト → △（柑橘の香りがロースト香に負ける可能性）
2. **AROMA / BITTER / SOUR / SWEET の記述**: 味覚への影響が醸造学的に正確か
   - 「苦味が増す」「酸味が加わる」などの記述が食材の成分と一致するか
3. **醸造工程の記述**: 副原料の投入タイミング（煮沸時/発酵時/熟成時）に関する記述があれば正確か
4. **専門用語の正確性**: ビアスタイル名、醸造用語のスペル・意味が正しいか

**不正確な情報が見つかった場合**: Phase 2に戻って該当箇所を修正する。

### Phase 5: ファクトチェック

投稿に含まれるすべての事実情報を検証する。**根拠のない数字・事実は絶対に使わない。**

**チェック項目**:
1. **数値データ**: 糖度、生産量、シェア率、標高、面積などの数字
   - WebSearchで裏取りする。検索結果と一致しない場合は修正または削除
   - 出典が見つからない数字は使わない
2. **地理情報**: 地域名、島名、山名、川名が正しいか
   - 例: 「生口島のレモン」→ 広島県尾道市に属する島で実際にレモン栽培が盛ん ✅
3. **食材の産地情報**: その県/地域が実際にその食材の名産地か
   - 例: 「富良野のメロン」→ 北海道富良野市は実際に赤肉メロンの名産地 ✅
4. **文化・歴史的記述**: 食材や土地に関する文化的な記述が事実に基づいているか
5. **既存ビールに関する記述**: 「柚子のビールは存在する」等の記述が事実か

**検証方法**:
- 数値や事実を含む文が1つでもあれば、WebSearchで検索して裏取りする
- 検証できない情報は「感覚的な描写」に書き換える（数字を使わない表現にする）
- 検証結果をチェックリストに記録する

**過去の失敗例（再発防止）**:
- ❌ メロンの糖度を「8.0」と記載 → 実際は12〜18度。根拠なく数字を使った
- ✅ 対策: 数字を使う場合は必ずWebSearchで確認してから記載する

### レビュー

出力後、以下をセルフチェックする:

**コピーライティング**:
- tone_guide.mdの禁止語が含まれていないか
- 「○○の人なら」等のよいしょ表現がないか
- スライド1が感情ファーストになっているか
- 各スライドの文字数が上限以内か
- 地域名を使った場合、スライド2で県名を回収しているか
- 投稿文に「県を一杯で表す」視点が含まれているか
- ハッシュタグが15〜20個の範囲か
- **全テキストに改行位置とY座標が明示されているか**

**ビール専門家チェック**:
- ビアスタイルと副原料の組み合わせが醸造学的に妥当か
- AROMA/BITTER/SOUR/SWEETの味覚記述が食材の成分と一致するか
- 醸造用語・スタイル名のスペルと意味が正しいか

**ファクトチェック（最重要）**:
- 投稿に含まれる数値（糖度、生産量、シェア率、標高など）をWebSearchで裏取りしたか
- 地理情報（地域名、島名）が正しいか
- 食材の産地情報が事実か
- **裏取りできない数値は使わない。感覚的な描写に置き換える**
- **根拠のない情報は絶対に発信しない**

## プロレベル運用基準

このスキルはROOTSのブランドを代表する公開コンテンツを生成する。
以下の基準を満たさない出力は、絶対に提出しない。

### 判断原則

1. **迷ったら出さない**: 事実か不明な情報、トーンに自信がない表現は削除する
2. **推測で埋めない**: 情報が足りない場合はユーザーに確認する。想像で補完しない
3. **数字は検証する**: すべての数値をWebSearchで裏取りする。検証できない数字は使わない
4. **一貫性を守る**: 前回の投稿と矛盾しないか、戦略ドキュメントと齟齬がないか確認する

### ハッシュタグルール（確定）

Instagramの運用上、**1投稿あたりハッシュタグは5個まで**。

構成:
- 発見用2個（#クラフトビール, #ビール好きな人と繋がりたい 等）
- 県名1個（#北海道, #福岡 等）
- ブランド用2個（#ルーツビール, #親愛なる故郷へ）

### エッジケース対応

| 状況 | 対処 |
|------|------|
| 食材名が長い（シークヮーサー等） | S1テキストの文字数を最優先でチェック。超えたら改行を増やす |
| スタイル名が長い（BELGIAN WHITE等） | Bebas Neueのサイズを64pxに縮小 |
| 県名と地域名の判断が難しい | instagram-strategy.mdの地名ルールに従う |
| 食材のビール相性が不明 | WebSearchで「[食材名] クラフトビール」で検索して実例を確認 |
| 投稿文が長くなりすぎた | キャプションは最大200文字を目安。超えたら削る |

### 過去の失敗ログ

新しい失敗が発生したらここに追記する:

1. **糖度8.0事件**: メロンの糖度を根拠なく「8.0」と記載。実際は12〜18度。→ 数字は必ずWebSearchで裏取り
2. **畑の矛盾事件**: 「畑を歩くと」と書いたがメロンはハウス栽培が主流。→ 栽培方法をファクトチェックの対象に追加
3. **テキスト影事件**: 写真の上にテキスト影を黒文字重ねで実装して安っぽくなった。→ グラデーション暗幕に変更
