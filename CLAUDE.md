# ROOTS — プロジェクト全体指示書

## このファイルについて
これはClaude Codeがプロジェクトを開いたときに最初に読む指示書です。
すべての作業は、ここに書かれたルールに従ってください。

---

## 事業概要

- 事業名：ROOTS
- コンセプト：都道府県民が副原料を投票で選び、OEM醸造されたビールをECで販売する参加型クラフトビール事業
- キャッチコピー：「親愛なる故郷へ」
- ターゲット：日本人（出身地を離れた20〜40代中心）
- 現フェーズ：Phase 1（MVP期）
- 長期目標：売上日本一のビール会社 / IPO・M&A視野

## デザインルール（変更禁止）

### フォント
| 用途 | フォント名 |
|------|-----------|
| 英語見出し・ロゴ | Bebas Neue |
| 日本語本文・和の品格 | Shippori Mincho |
| コード・数値・機能的テキスト | Space Mono |

### カラーパレット
| 名前 | コード | 用途 |
|------|--------|------|
| 背景（黒） | #090909 | 全媒体共通の背景色。白背景は禁止 |
| ブルー | #50A0F0 | メインアクセント、見出し、リンク、装飾ライン |
| ブルー（明） | #60B0FF | グラデーション用、補助的なアクセント |
| オレンジ | #F0A032 | CTA、ボタン、強調テキスト、ロゴ |
| オレンジ（暖） | #FF8030 | グラデーション用、暖色アクセント |
| テキスト（明） | #e0e0e0 | メインテキスト |
| テキスト（暗） | #333333 | 補足テキスト |

### グラデーション
- メイングラデーション: `linear-gradient(135deg, #60B0FF 0%, #FFC050 50%, #FF8030 100%)`
- ブルー→オレンジ: `linear-gradient(90deg, #50A0F0, #F0A032, #FF7020)`

### 絶対ルール
- 背景は常に #090909。白背景のデザインは作成しないこと
- ラベルデザインにはレッド系の色を使わないこと
- フォントは上記3種のみ使用すること

## フォルダ構成と役割

```
ROOTS/
├── docs/       … 全社確定ドキュメント（事業計画書など）
├── web/        … Web開発（HP・投票システム）
├── brand/      … ブランド（素材・ラベル）
├── biz/        … ビジネス（免許・OEM・EC）
└── marketing/  … マーケティング（SNS・ローンチ）
```

各領域フォルダにはAGENTS.mdがあります。
そのフォルダ内で作業する前に必ず読んでください。

## Git運用ルール

- 作業前にコミット（セーブポイント）を作ること
- コミットメッセージは日本語で、何をしたか簡潔に書く
  - 例：`HP: ヒーローセクション実装`
  - 例：`biz: OEM候補リスト更新`
- 大きな変更の前にはブランチを切ることを検討する
