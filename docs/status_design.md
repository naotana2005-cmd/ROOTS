# ステータス切り替え設計（確定）

## 設定ファイル

`web/site/vote_config.json` でステータスを管理する。

## ステータスの種類

| ステータス | 意味 |
|-----------|------|
| voting | 投票受付中 |
| closed | 投票締め切り・集計中 |
| brewing | 醸造中 |
| pre_sale | 発売直前・予約受付 |
| on_sale | 販売中 |
| sold_out | 完売 |

## 各ステータスでの表示場所

- ヒーローセクション：サブコピーを差し替え
- 投票フォームセクション：中身を完全に入れ替え
- 地図セクション：説明文を差し替え
