# ドキュメントエージェント指示書

## 担当範囲
- 全社確定ドキュメント（事業計画書、プレゼン資料など）
- 確定した仕様書・方針書

## 守るべきルール
- デザインルールはCLAUDE.mdが正（このフォルダではなくCLAUDE.mdを参照）
- 個人情報（住所・氏名・電話番号など）を含むファイルはこのフォルダに保存しないこと
- ドラフト・メモ・アイデアはNotionに置く（ここには確定版のみ）
- フォントは Bebas Neue / Shippori Mincho / Space Mono のみ

## ファイル命名規則
- `[カテゴリ]-[内容].[拡張子]`
  - 例：`plan-business-v1.pdf`
  - 例：`spec-voting-system.md`
  - 例：`pitch-deck-v2.pptx`
