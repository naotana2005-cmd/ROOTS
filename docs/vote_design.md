# 投票設計（確定）

## 投票方式

- 自由記述・1回のみ・二次投票なし

## 副原料の決定ルール

- 得票数トップの素材を採用
- 製造不可能な場合は得票順に繰り下げ
- このルールはサイトに公開して透明性を担保

## 副原料の線引き

### OK

- 野菜・果物・海産物（昆布・鰹節等の乾燥素材含む）
- きのこ・山菜・茶葉
- 穀物・ナッツ類

### NG

- 調理済み食品
- 調味料
- 菓子
- 肉類

## サジェスト表示

- 県選択後にその県の農産物を3〜5個カード表示
- カードをタップすると入力欄に自動入力
- 「自分で書く」ボタンで自由記述に切り替え可

## 北海道の扱い

- 地方選択で「北海道」を選んだ場合は
  都道府県選択をスキップしてそのまま確定
