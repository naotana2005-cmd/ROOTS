# ROOTS

**「親愛なる故郷へ」**

都道府県民が副原料を投票で選び、OEM醸造されたビールをECで販売する参加型クラフトビール事業。

## フォルダ構成

| フォルダ | 内容 |
|---------|------|
| `docs/` | 事業計画書など全社ドキュメント |
| `web/` | ホームページ・投票システムのコード |
| `brand/` | ロゴ・ラベル・デザイン素材 |
| `biz/` | 免許申請・OEM選定・EC準備 |
| `marketing/` | SNS・ローンチ戦略 |

## 運用ルール

- **コード・素材・確定文書** → このフォルダ
- **タスク管理・進捗・メモ** → Notion
- AIエージェントへの指示は各フォルダの `AGENTS.md` に記載

## 開発環境

Claude Codeをメインの開発ツールとして使用。
プロジェクト全体の指示は `CLAUDE.md` に記載。
