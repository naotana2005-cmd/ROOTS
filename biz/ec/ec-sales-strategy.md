# 販売チャネル・価格設計戦略

最終更新：2026-03-19

---

## 1. 結論（推奨方針）

- **販売モデル**: セット固定型 ＋ 送料込み価格
- **セット構成**: 4本 / 6本 / 12本 の3種のみ（箱も3種のみ）
- **ローンチ手段**: クラウドファンディング（Makuake or CAMPFIRE）で先行販売
- **中期チャネル**: ふるさと納税（都道府県コンセプトと直結）
- **価格心理**: 松竹梅の法則で6本セットに誘導

## 3. 箕面ビール型（1本単位販売）を採用しない理由

箕面ビールは1本単位で購入でき、24本まで1個口で地域別送料がかかる仕組み。
自由度が高い反面、ROOTSには合わない。

| 課題 | 内容 |
|------|------|
| 箱のバリエーション | 1〜24本に対応する箱・仕切り・緩衝材が必要。小ロット事業にはオペレーション負荷が大きすぎる |
| 梱包コスト | 本数ごとに最適な箱を選ぶ手間。大箱＋緩衝材で吸収すると見栄えが悪くブランド毀損 |
| 送料の見え方 | 「商品550円 ＋ 送料1,067円」→ 送料が商品の2倍。購買心理に大きなマイナス |
| ROOTSとの不一致 | ROOTSは1県1種類。「1本だけ試す」ニーズより「セットで味わう」体験のほうが合う |

**結論**: ROOTSはセット固定型を採用し、箱を3種類に限定する。

## 4. ROOTSの販売モデル設計

### 4.1 セット構成（確定方針）

| セット名 | 本数 | 価格（税込・送料込み） | 役割 | 箱サイズ |
|---------|------|---------------------|------|---------|
| お試しセット | 4本 | 2,980円 | 初回購入のハードルを下げる | 60サイズ |
| **基本セット** | **6本** | **4,980円** | **メイン商品。利益の柱** | **60サイズ** |
| たっぷりセット | 12本 | 8,980円 | リピーター・まとめ買い | 80サイズ |

> **⚠️ 価格未確定（2026-03-22更新）**
> カウボーイクラフトの料金表を分析した結果、缶OEMの1本あたり原価が~710円となり、
> 6本セット4,980円（送料込み）ではクール便の場合赤字になることが判明。
> 瓶（クラフトハート: ~488円/本）なら4,980円で粗利24%が出る。
> **醸造所の見積もり比較が完了するまで、最終価格は確定しない。**
> 4,980円を維持するための1本あたり原価の条件:
> - 損益分岐: **688円/本以下**（酒税込み）
> - 粗利15%: **564円/本以下**（酒税込み）
> 詳細は `../oem/packaging-strategy.md` セクション6を参照。

### 4.2 なぜこの3段階か（松竹梅の法則）

消費者は3つの選択肢を提示されると**真ん中を選びやすい**。

```
  お試し（4本）   →   基本（6本）   →   たっぷり（12本）
     2,980円            4,980円            8,980円
                           ↑
                    ここに誘導される
```

- 4本: 「安いけど少ない」→ もう少し出して6本にしよう
- 12本: 「お得だけど多い」→ まずは6本で試そう
- **6本が「ちょうどいい」と感じる設計**

### 4.3 箱の運用

**箱は3種類のみ。**

| 箱 | 対象セット | サイズ | 単価目安 | 仕様 |
|----|----------|--------|---------|------|
| 箱A | 4本セット | 60サイズ | 100〜150円 | 350ml缶4本用。仕切り不要（缶なので） |
| 箱B | 6本セット | 60サイズ | 130〜200円 | 350ml缶6本用。ギフト対応可 |
| 箱C | 12本セット | 80サイズ | 150〜250円 | 350ml缶12本用。6本×2段 |

缶は瓶と違い割れないため、仕切り・緩衝材が最小限で済む。
オリジナル段ボール（ROOTSロゴ入り）はPhase 2以降で検討。初期は既製品で十分。

## 5. 送料設計

### 5.1 「送料込み」を採用する理由

| 方式 | 顧客の見え方 | 購買心理 |
|------|------------|---------|
| 送料別（箕面型） | 「商品550円 ＋ 送料1,067円」 | 送料を見てカート離脱しやすい |
| **送料込み** | **「4,980円（送料無料）」** | **追加コストなし。安心して購入できる** |

「送料無料」はEC購買における最強のコンバージョン促進要素。
実際の送料（法人契約で600〜800円/件）は商品価格に含めて吸収する。

### 5.2 地域別の送料差をどう吸収するか

ヤマト法人契約後も、北海道・沖縄は送料が高い。

| 方法 | 内容 | 採用判断 |
|------|------|---------|
| 全国一律で吸収 | 北海道・沖縄の送料増を全体に薄く転嫁 | ○ シンプルだが北海道・沖縄比率が増えると赤字 |
| **北海道・沖縄のみ追加送料** | 「＋500円」などの追加料金 | **◎ 推奨。多くのECが採用** |
| 離島は配送不可 | 離島を除外 | △ 機会損失 |

**推奨**: 本州・四国・九州は送料込み。北海道・沖縄のみ追加送料（＋500〜800円）を設定。

## 6. 販売チャネル戦略（Phase別）

### Phase 1: クラウドファンディングでローンチ

**なぜクラファンか**

| メリット | 内容 |
|---------|------|
| 在庫リスクゼロ | 受注してから醸造。売れ残りなし |
| 需要の事前検証 | 何人が買うかがローンチ前にわかる |
| PR効果 | メディア掲載・SNS拡散が期待できる |
| 初期顧客獲得 | 支援者がそのまま初期ファンになる |
| ストーリーとの親和性 | 「投票で副原料を選んだビール」はクラファンのストーリーに最適 |

**プラットフォーム比較**

| 項目 | Makuake | CAMPFIRE |
|------|---------|----------|
| 手数料 | 20%（決済手数料込み） | 17%（決済手数料込み） |
| ユーザー層 | 30〜40代男性が多い | 幅広い |
| ビール案件 | 50件以上の実績 | 1,000件以上の実績 |
| 特徴 | 「応援購入」のブランド力が高い | 手数料が安い。ビール専門担当がいる |
| ROOTSとの相性 | ◎ ターゲット層が一致 | ○ 手数料重視なら |

**クラファンでの販売設計（案）**

| リターン | 内容 | 価格 | 限定数 |
|---------|------|------|--------|
| 早割お試し | 4本セット | 2,480円 | 50セット |
| 基本セット | 6本セット | 4,480円 | 100セット |
| まとめ買い | 12本セット | 7,980円 | 50セット |
| 応援プラン | 6本＋ROOTSオリジナルグラス | 6,980円 | 30セット |

※クラファン手数料（17〜20%）を考慮し、通常販売価格より割安に設定

### Phase 2: 自社ECサイト ＋ ふるさと納税

**自社EC（Shopify / BASE / STORES）**

クラファン終了後、自社ECで通常販売を開始。

| プラットフォーム | 月額 | 決済手数料 | 特徴 |
|----------------|------|----------|------|
| Shopify | 約4,000円〜 | 3.4%〜 | カスタマイズ性高い。デザイン自由度◎ |
| BASE | 0円 | 6.6%＋40円 | 無料で始められる。手数料が高め |
| STORES | 0円〜 | 5.0%〜 | シンプル。決済手数料が中間 |

※ECプラットフォームの詳細比較は別途 `ec-platform-comparison.md` で行う

**ふるさと納税**

ROOTSの都道府県コンセプトとの親和性が**極めて高い**。

| メリット | 内容 |
|---------|------|
| 送料は自治体が負担 | ROOTSの送料コスト問題が解消される |
| 県コンセプトと直結 | 「大阪のROOTSビール」→ 大阪の自治体に提案 |
| 還元率の自由度 | クラフトビールは50%超の還元率も可能（大手は約30%固定） |
| 新規顧客の獲得 | ふるさと納税ユーザーからの流入 |

**実現に必要なこと（※要確認）**:
- OEM醸造所の所在地の自治体に返礼品として登録できるか
- ROOTSの事業所所在地との関係
- 返礼品登録の手続き・審査基準
- 納品スケジュールとの整合

### Phase 3: サブスクリプション＋追加チャネル

複数都道府県の展開後に以下を追加:

**サブスク「毎月届く、ふるさとのビール」（仮称）**
- 月額4,980〜5,480円
- 毎月違う都道府県のビール6本が届く
- 送料込み
- スキップ無料・いつでも解約可

**参考モデル**:
- よなよな「月の生活」: 12本/月〜、送料無料、会報誌付き
- Otomoni: 6本/回 4,980円、4,000種以上から厳選
- 伊勢角屋「追いビール」: 定期会員が追加購入時に送料920円OFFクーポン

## 7. 価格心理の活用

### 7.1 ROOTSで使える手法

| 手法 | ROOTSでの活用 |
|------|-------------|
| **松竹梅の法則** | 4本/6本/12本の3段階。6本に自然と誘導される |
| **送料込み表示** | 「送料無料」で購買ハードルを下げる |
| **初回限定価格** | クラファンの早割、または自社ECの初回限定クーポン |
| **追いビール制度** | リピーターに「次回送料500円OFF」クーポンを発行 |
| **ストーリー価値** | 「あなたの投票がビールになった」→ 価格以上の感情的価値を提供 |
| **ギフト訴求** | 「同郷の友人に贈る」需要。ギフト包装＋メッセージカード対応 |

### 7.2 アップセル設計

```
カート画面のイメージ:

┌────────────────────────────────┐
│  基本セット（6本） 4,980円      │
│                                  │
│  ┌──────────────────────────┐  │
│  │ たっぷりセット（12本）なら   │  │
│  │ 1本あたり249円お得！          │  │
│  │ → 12本セットに変更する       │  │
│  └──────────────────────────┘  │
└────────────────────────────────┘
```

1本あたり単価の比較:
- 4本セット: 745円/本
- 6本セット: 830円/本
- 12本セット: **748円/本** ← 4本と同水準で量が3倍

※6本セットの1本単価が最も高く見えるが、「送料込みで最も手軽」という価値で補う

### 7.3 リピーター施策

| 施策 | 内容 | 導入時期 |
|------|------|---------|
| 追いビールクーポン | 購入後に「次回送料500円OFF」クーポン配布 | Phase 1〜 |
| 投票参加者限定割引 | 投票した人だけが使える初回500円OFFコード | Phase 1〜 |
| レビュー特典 | レビュー投稿で次回100円OFF | Phase 2〜 |
| サブスク | 月額固定で毎月届く | Phase 3〜 |

## 9. リスクと対策

| リスク | 対策 |
|--------|------|
| クラファンで目標未達 | 目標金額を低めに設定（50万円程度）。All-in方式を選択 |
| 送料込み価格で利益が出ない | パストライズ常温便でコスト削減。北海道・沖縄は追加送料 |
| ふるさと納税の登録が困難 | OEM醸造所の所在地自治体を候補に。複数自治体に打診 |
| 4本お試しセットで赤字 | 限定数を設定（50〜100セット）。顧客獲得コストと割り切る |
| リピート率が低い | 追いビール制度＋次回投票への導線を設計 |
