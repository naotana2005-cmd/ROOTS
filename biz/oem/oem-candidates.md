## 1. 醸造所選定の進め方

### 前提（確定方針）

ここまでの戦略検討で、以下が決まっている。
醸造所選定はこれらの方針を踏まえて進める。

| 項目 | 確定方針 | 参照 |
|------|---------|------|
| 容器 | **350ml缶**（フォールバック: 330ml瓶） | packaging-strategy.md |
| ラベル | **シュリンクラベル**（無地缶に全面フィルム） | packaging-strategy.md |
| セット | **4本/6本/12本** の3種のみ | ec-sales-strategy.md |
| 価格 | 6本セット **4,980円**（送料込み） | ec-sales-strategy.md |
| 配送 | 初期クール便 → パストライズ対応なら常温便 | packaging-strategy.md |
| ローンチ | **クラウドファンディング**で先行販売 | ec-sales-strategy.md |
| 発送 | 初回は**醸造所から直送** → 軌道に乗ったら3PL | legal_memo.md |

### 醸造所に求める「必須条件」と「加点条件」

上記の方針から、醸造所に求める条件を2段階に整理する。

**必須条件（これを満たさない醸造所は候補から外す）**
- [ ] OEM委託醸造に対応している
- [ ] 品目ごとの年間課税移出数量が3,000kl未満（免許要件）
- [ ] 副原料（農産物・海産物・加工品等）を使った醸造に対応可能
- [ ] 小ロット対応（1,000本以下）
- [ ] エール系スタイルの醸造実績がある

**加点条件（優先度を上げる要素）**
- [ ] **缶充填機を保有**（缶が第一希望のため。最重要の加点条件）
- [ ] **パストライズ（熱処理）対応可**（常温配送→送料大幅削減）
- [ ] ビール＋発泡酒の両免許（副原料の使用範囲が広がる）
- [ ] 副原料の加工（殺菌・乾燥等）を醸造所側で対応してくれる
- [ ] ラベル貼り・梱包・発送まで一貫対応（初期は醸造所直送のため重要）
- [ ] テストバッチ（30〜200L）対応
- [ ] 品川から1時間圏内
- [ ] レシピ権利がROOTS側に帰属する契約が可能

---

### 全体スケジュール（目安：約3〜4ヶ月）

```
STEP 1           STEP 2           STEP 3          STEP 4
一次コンタクト → 訪問・テスト醸造 → 契約交渉 → クラファン準備＋初回醸造
 (2週間)          (3〜4週間)       (2〜3週間)     (4〜6週間)
      ├─並行─┐
      │ threefeet相談 │
      └──────┘
```

**旧ステップとの違い**:
- 旧STEP 2「テスト醸造の相談」と旧STEP 3「訪問・試飲」を**統合**。実際は訪問時にテスト醸造の話もするため
- 新STEP 4に**クラファン準備**を追加。醸造所確定とクラファン準備は並行して進める
- 一次コンタクトで聞く項目に**缶充填・パストライズ・発送対応**を追加

---

### STEP 1: 一次コンタクト（2週間）

**やること**

| 優先度 | アクション | 対象 |
|--------|----------|------|
| 最優先 | メール送付（缶充填＋テストバッチ） | **カウボーイクラフト**（Tier S、缶充填機あり） |
| 最優先 | メール送付（副原料対応＋両免許） | **クラフトハート**（Tier S、瓶フォールバック最有力） |
| 最優先 | メール送付（副原料実績＋品質） | **サンクトガーレン**（Tier S） |
| 最優先 | メール送付（副原料実績＋近さ） | **T.T BREWERY**（Tier S、品川20分） |
| 同時 | コンサル相談 | **threefeet 多田氏**（缶充填対応の醸造所を紹介してもらう） |
| 返答待ち | 軽めの問合せ | IB BREWING（缶充填、ただしレシピ権利に注意） |

**メールで確認すべき項目（全16項目）**

醸造の基本:
- [ ] OEM委託醸造の対応可否
- [ ] 最小ロット数（何Lから / 何本から）
- [ ] 1本あたりの単価目安（350ml缶 or 330ml瓶）
- [ ] エール系の醸造実績・得意スタイル
- [ ] 醸造〜納品までのリードタイム

副原料:
- [ ] 副原料の持ち込み可否と衛生基準
- [ ] 副原料の前処理（殺菌・乾燥等）は醸造所側で対応可能か
- [ ] 毎回異なる副原料に対応できるか（柔軟なレシピ開発体制の有無）

容器・包装（※新規追加項目）:
- [ ] **缶充填への対応可否**（350ml缶。充填機の種類）
- [ ] **パストライズ（加熱殺菌）への対応可否と費用**
- [ ] **シュリンクラベル貼りへの対応可否**（醸造所側で巻けるか）
- [ ] **梱包・発送への対応可否**（完成品を段ボールに詰めて顧客に直送できるか）

免許・書類:
- [ ] 品目ごとの年間課税移出数量が3,000kl未満であることの確認
- [ ] 「取引承諾書」「仕入見込み証明書」の発行可否

テスト醸造:
- [ ] テストバッチ（30〜200L）の対応可否と費用
- [ ] テスト醸造から本発注に移行する場合の流れ

**判断基準（3段階で評価）**

| 評価 | 基準 |
|------|------|
| ◎ 即次ステップへ | 缶充填可 ＋ 副原料柔軟 ＋ 発送対応可 ＋ レスポンスが速い |
| ○ 継続検討 | 瓶のみだが副原料・コスト・品質のいずれかに強みがある |
| △ 保留 | 返信が遅い、条件が合わない、確認事項が多すぎる |

### STEP 2: 訪問・テスト醸造（3〜4週間）

**やること**
- STEP 1で◎〜○の評価を得た醸造所（目安: 2〜3社）を訪問
- 訪問時に既存ビールを試飲し、品質・味の方向性を確認
- テスト醸造の具体的な条件をその場で詰める

**訪問時チェックリスト（22項目）**

品質・設備:
- [ ] 設備の規模感と清潔さ
- [ ] HACCP等の品質管理体制
- [ ] 缶充填機の種類と状態（缶対応の場合）
- [ ] パストライズ設備の有無
- [ ] 冷蔵保管のスペース

コンセプト理解:
- [ ] 担当者がROOTSのコンセプト（毎回異なる副原料）を理解しているか
- [ ] 「投票→醸造→販売」の3ヶ月サイクルに対応できるか
- [ ] レシピ開発にどの程度協力してくれるか（提案力があるか）

容器・包装・発送:
- [ ] 缶充填の具体的な流れ（充填→密封→パストライズの工程）
- [ ] シュリンクラベルの装着は誰がやるか（醸造所 or 外部 or 自分）
- [ ] 梱包（段ボール詰め）は醸造所で対応できるか
- [ ] 完成品の保管期間と費用
- [ ] 顧客への直送（醸造所→個人宅）は対応可能か
- [ ] ヤマト運輸等との集荷契約は醸造所側にあるか

コスト:
- [ ] テストバッチの費用（30〜200L）
- [ ] 本発注時の1本あたり単価（缶代・充填代込み）
- [ ] ラベル貼りの費用（醸造所対応の場合）
- [ ] 梱包・発送の費用（醸造所対応の場合）
- [ ] 前払い・後払い・分割の条件

権利・契約:
- [ ] レシピの権利帰属（ROOTS側に帰属させたい）
- [ ] 醸造所名の表示方法（酒税法上の義務）
- [ ] テスト醸造の結果が気に入らなかった場合のキャンセルポリシー

**テスト醸造の進め方**

1. 副原料を1つ決める（例: 東京都なら小笠原パッションフルーツ）
2. 醸造所と味の方向性をすり合わせる（対面 or リモート）
3. テストバッチ醸造（30〜200L）
4. 試飲して評価
5. 問題なければ本発注の条件交渉（STEP 3）へ

**この段階で「缶 or 瓶」を最終確定する。**
- カウボーイクラフトまたはIB BREWINGの缶充填が条件的にOK → 缶で確定
- 缶が不可 → クラフトハートの瓶にフォールバック

---

### STEP 3: 契約交渉（2〜3週間）

**やること**
- 1社（＋予備1社）に絞り、契約条件を交渉
- 免許申請に必要な書類を発行してもらう

**契約で確定すべき事項（18項目）**

免許関連（最重要）:
- [ ] 年間課税移出数量3,000kl未満の**書面での確認**
- [ ] 「取引承諾書」の発行
- [ ] 「仕入見込み証明書」の発行

製造条件:
- [ ] 1回の醸造ロット（何L / 何本）
- [ ] 1本あたりの製造単価（醸造＋缶充填＋パストライズの内訳）
- [ ] 副原料の持ち込みルール（衛生基準、前処理の要否）
- [ ] 副原料が醸造に適さない場合のフロー
- [ ] 醸造〜納品までのリードタイム（3ヶ月サイクルに乗るか）

包装・発送:
- [ ] シュリンクラベル貼りの対応可否と費用
- [ ] 梱包（段ボール詰め）の対応可否と費用
- [ ] 個人宅直送の対応可否と1件あたりの費用
- [ ] 完成品の保管条件と期間

権利・責任:
- [ ] **レシピの権利帰属**（ROOTS側に帰属させる条項を入れる）
- [ ] 製造物責任（PL）の所在
- [ ] ラベルデザインの権利はROOTS側にあることを明記
- [ ] 醸造所名の表示方法（酒税法上の義務）

リスク対応:
- [ ] キャンセルポリシー（副原料が調達できなかった場合）
- [ ] 納期遅延時の対応方針
- [ ] 品質基準の定義（アルコール度数の誤差範囲、異物混入時の対応）

支払い:
- [ ] 支払い条件（前払い・後払い・分割）
- [ ] テスト醸造費用は本発注に充当されるか

**※注意**: IB BREWINGはOEM終了後にレシピ所有権が醸造所側に帰属するルールあり。契約前に必ず確認し、ROOTS側帰属に変更できなければ候補から外す。

---

### STEP 4: クラファン準備＋初回醸造（4〜6週間）

醸造所が確定したら、**クラウドファンディングの準備**と**初回醸造**を並行して進める。

```
     ┌─── クラファン準備 ──────────────────┐
     │ プラットフォーム選定                    │
     │ プロジェクトページ作成（写真・動画・文章） │
     │ リターン設計（4本/6本/12本＋特典）       │
     │ 目標金額の設定                          │
     └──────────────────────────────────┘
                    ↕ 並行
     ┌─── 初回醸造 ────────────────────────┐
     │ 第1県の副原料を決定                     │
     │ 副原料の調達・農家交渉                   │
     │ 醸造所に副原料を送付                     │
     │ 醸造・熟成（エール系: 3〜4週間）          │
     └──────────────────────────────────┘
                    ↕ 並行
     ┌─── ラベル・梱包準備 ──────────────────┐
     │ シュリンクラベルのデザイン最終化           │
     │ ラベル印刷会社への発注                   │
     │ 段ボール3種（4本/6本/12本用）の発注       │
     │ ヤマト運輸の法人契約締結                  │
     └──────────────────────────────────┘
```

**クラファンの販売設計**

| リターン | 内容 | 価格 | 限定数 |
|---------|------|------|--------|
| 早割お試し | 4本セット | 2,480円 | 50セット |
| 基本セット | 6本セット | 4,480円 | 100セット |
| まとめ買い | 12本セット | 7,980円 | 50セット |
| 応援プラン | 6本＋ROOTSオリジナルグッズ | 6,980円 | 30セット |

※詳細は `ec-sales-strategy.md` を参照

**初回醸造のタイムライン**

```
Week 1-2  : 副原料の決定・農家交渉・調達
Week 3    : 副原料を醸造所に送付
Week 3-6  : 醸造・熟成（エール系: 3〜4週間）
Week 5-6  : ラベル貼り・梱包（醸造と並行して準備）
Week 7    : クラファン公開 → 受注開始
Week 8-9  : 発送開始（醸造所から直送）
```

---

### STEP 5: 本格運用（3ヶ月サイクル）

クラファンが成功し、自社ECサイトが立ち上がったら、以下の3ヶ月サイクルで運用する。

```
月1        : 投票期間（1ヶ月）
月2 前半   : 副原料決定・農家交渉（2週間）
月2 後半   : 副原料送付 → 醸造開始
月2〜月3   : 醸造・熟成（3〜4週間、エール系）
月3 並行   : ラベル発注・EC商品ページ準備
月3 末     : ラベル貼り・梱包 → 発売開始
```

### 意思決定フローチャート

```
缶充填対応の醸造所にコンタクト
  │
  ├─ カウボーイクラフトが条件OK?
  │    ├─ YES → 缶で確定。テスト醸造へ
  │    └─ NO ──┐
  │              │
  ├─ IB BREWINGが条件OK?（レシピ権利含む）
  │    ├─ YES → 缶で確定。テスト醸造へ
  │    └─ NO ──┐
  │              │
  ├─ threefeetから缶対応の醸造所を紹介された?
  │    ├─ YES → 条件確認 → テスト醸造へ
  │    └─ NO ──┐
  │              │
  └─ 缶を断念 → 瓶にフォールバック
       │
       ├─ クラフトハートが条件OK?
       │    ├─ YES → 瓶で確定。テスト醸造へ
       │    └─ NO → T.T BREWERY or サンクトガーレンへ
       │
       └─ 瓶の醸造所を1社確定 → テスト醸造へ
```

## 2. 候補醸造所一覧（全20社・ティア分け）

### 評価基準

| 評価軸 | 内容 |
|--------|------|
| 副原料対応 | 多様な副原料への柔軟性・実績 |
| 小ロット | MVP期に適した少量対応力 |
| アクセス | 品川からの所要時間 |
| OEM明示 | 公式サイトでOEM対応を明記しているか |
| 免許 | ビール＋発泡酒の両免許があるか（副原料の幅に影響） |
| コスト | 初期投資の現実性 |

### Tier S — 最優先コンタクト（MVP期に最適）

#### ① クラフトハート・ブルワリー
- **所在地**: 千葉県佐倉市（品川から約80分）
- **最小ロット**: 200L（約550本/330ml瓶）
- **OEM**: 明示あり。実績30万本超
- **副原料**: 野菜・果物・海産物OK。素材を送れば醸造所側で加工
- **免許**: ビール＋発泡酒の両方
- **得意スタイル**: エール主体
- **単価目安**: 約375円/本〜（330ml瓶・副原料持込時）
- **製造期間**: 約2ヶ月（打合せ1ヶ月＋醸造1ヶ月）
- **URL**: https://oem.craft-heart.com/
- **選定理由**: 副原料を送るだけで加工してくれる体制、ビール＋発泡酒の両免許、明確な価格帯。ROOTSの「毎回異なる副原料」に最も適合する

#### ② T.T BREWERY
- **所在地**: 神奈川県川崎市（品川から約20分）
- **最小ロット**: セミオーダー18L（樽1本）/ フルオーダー180L（樽10本）
- **OEM**: 明示あり
- **副原料**: 野菜・海産物・山芋・茶・桜・かぼちゃ・コーヒーなど多数。地域特産品ビールの実績豊富
- **得意スタイル**: エール系全般
- **単価目安**: 開発費22,000円（初回のみ）＋醸造費18,150円/樽（18L）
- **URL**: https://ttbrewery.jp/oem/
- **選定理由**: 品川から最も近いクラスのアクセス、18Lからのテスト醸造が可能、地域特産品ビールの実績が豊富でROOTSのコンセプトに直結

#### ③ サンクトガーレン
- **所在地**: 神奈川県厚木市（品川→本厚木 約60分）
- **最小ロット**: 要問合せ
- **OEM**: パークハイアット東京、タワーレコード等に供給実績
- **副原料**: 桜・チョコレート・湘南ゴールド・梨・桃・リンゴ等。副原料ビールの日本トップランナー
- **免許**: ビール＋発泡酒
- **得意スタイル**: エール専門（1997年〜）
- **URL**: https://www.sanktgallenbrewery.com/
- **選定理由**: 日本クラフトビール界のパイオニア。副原料ビールの品質・実績は最高水準。ブランド力がROOTSの信頼性向上に寄与。小ロット対応の可否が要確認

#### ④ カウボーイクラフト ブルワリー
- **所在地**: 神奈川県小田原市（品川から約75分）
- **最小ロット**: 30L〜（テストバッチ）← 業界最小クラス
- **OEM**: 明示あり
- **副原料**: 要相談
- **設備**: 缶充填機あり（Wild Goose Filling社製）
- **URL**: https://www.cowboycraftbrewing.com/
- **選定理由**: 30Lテストバッチで味を確認してから本発注できる。缶対応はEC販売に有利。MVP期のリスク最小化に最適

### Tier A — 有力候補（条件次第で採用したい）

#### ⑤ VECTOR BREWING
- **所在地**: 東京都台東区浅草橋 / 新宿区（品川から約30分）
- **最小ロット**: 100L〜
- **OEM**: 明示あり
- **得意スタイル**: 多様（非加熱・無濾過が基本）
- **URL**: https://vectorbrewing.co.jp/category/vectorbrewing/oem-odm
- **備考**: 都内2拠点。100Lからの小ロット対応

#### ⑥ OGA BREWING
- **所在地**: 東京都三鷹市（品川から約40分）
- **最小ロット**: 300L（1仕込み）
- **OEM**: 明示あり
- **副原料**: 150種以上のレシピ開発実績
- **URL**: https://oga-brewing.com/
- **備考**: 元デザイナーが運営。攻殻機動隊・おそ松さん等の有名IPコラボ実績。ブランディング理解が深い

#### ⑦ Y.Y.G. BREWING COMPANY
