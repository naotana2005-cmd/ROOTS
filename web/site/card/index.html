カードの見方
レア度（N / R / SR / SSR）
出身地と現在地の距離で決まります。同県はN、隣接県はR、中距離はSR、最遠距離はSSRです。
ルーツ度
故郷との距離の近さ。地元に住んでいる人ほど高く、同県在住なら100。
郷愁力
故郷を想う力。ふとした瞬間に地元を思い出す度合い。
冒険度
故郷からどれだけ遠くに来たか。距離が遠いほど高い。
地元愛
同じ県から登録した仲間の多さ。登録者が多い県ほど高い。
