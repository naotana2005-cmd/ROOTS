# Web開発エージェント指示書

## 担当範囲
- `site/` … ホームページ本体
- `vote/` … 投票システム（Supabase連携）

## 技術方針
- HP構成：8セクション構成（仕様確定済み）
- 投票データ管理：Supabase
- 地図UI：D3.js
- 投票後AI返答：Claude API

## 守るべきルール
- デザインルールはCLAUDE.mdを参照すること（カラー・フォント）
- 背景は必ず #090909
- CTAボタンのカラーはWebでは #d4412a（レッド）を使用可
- フォントは Bebas Neue / Shippori Mincho / Space Mono のみ

## フォルダの使い方
- `site/` にHP関連のコードをすべて配置
- `vote/` に投票システム関連のコードを配置
- site と vote は将来別々にデプロイする可能性があるため、依存を最小限にすること
